    2. ui-materialize

* Cassandra: 2.1.2
* Elasticsearch: 1.7.3

# Setup

* Install bower_components
  * Go to [root project] `/public`
  * `$ bower install`

* Install and turn on `casandra`
* Install and turn on `elasticsearch`

* Run project

```shell

$ activator run

```
